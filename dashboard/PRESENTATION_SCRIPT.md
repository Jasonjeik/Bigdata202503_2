about movies.

**The Problem**: Traditional movie feedback is either:
- Too slow (post-screening surveys)
- Too simple (star ratings only)
- Not actionable (no sentiment insights)

**Our Solution**: Real-time sentiment analysis powered by 4 advanced AI models, 
giving you instant, detailed insights into audience opinions."

**[Key Stats on Screen]**:
- 4 AI models with up to 91.6% accuracy
- Real-time analysis in milliseconds
- Visual, interactive dashboard

## Technical Deep Dive (If Asked)

**Architecture:**
- Frontend: Streamlit (Python)
- Database: MongoDB Atlas (Cloud)
- ML Models: PyTorch, Transformers, Scikit-learn
- Hosting: Streamlit Cloud / Azure
- API: RESTful design for extensibility

**Model Details:**
1. **DistilBERT**: 66M parameters, transformer architecture
2. **LSTM**: Bidirectional, 2 layers, optimized with Optuna
3. **Logistic Regression**: TF-IDF vectorization, GridSearchCV tuned
4. **Random Forest**: Ensemble of 100 trees

**Performance:**
- Response time: <500ms per prediction
- Database queries: <100ms average
- Concurrent users: 1000+ supported
